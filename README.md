## 🚀 Como Usar Localmente

1. **Clone o repositório:**
```bash
git clone https://github.com/joseevertoneverton92-maker/one-piece-brasil.git
cd one-piece-brasil
```

2. **Inicie um servidor local:**
```bash
# Python 3
python -m http.server 8000

# Ou Node.js
npx http-server
```

3. **Abra no navegador:**
```
http://localhost:8000
```
